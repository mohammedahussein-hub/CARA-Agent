# CARA: Cloud Requirements Analyzer Agent

This repository contains the source code and experimental results for the paper:

**"CARA: A Bilingual Agentic AI Framework for Cloud Requirements Analysis"**

## Results
- Controlled test (6 items): 83.33% accuracy
- English benchmark (19 items): 15 FR, 4 NFR (21.1% NFR)
- Arabic test set (8 items): 2 FR, 6 NFR (75.0% NFR)

## How to Use
Run `CARA_agent_final.py` in any Python environment.
